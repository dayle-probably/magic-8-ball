# Magic 8-Ball Discord Bot

A Discord bot that can see your fate. Responds to @ mentions and any message that starts with the prefix `!8`.

![magic 8-ball screenshot](magic-8-ball.png)

This is a basic [discord.js](https://github.com/discordjs/discord.js) template for a bot that can read messages in channels and DMs. Just add your bot token to `.env`.
